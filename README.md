## Instalación

1. Descarga todos los archivos en una carpeta
2. Abre `index.html` en cualquier navegador web moderno
3. Para desarrollo local, usa un servidor HTTP simple:
   ```bash
   # Con Python 3
   python -m http.server 8000
   
   # Con Node.js
   npx serve
   ```
